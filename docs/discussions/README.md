## 核心产品决策摘要

1. **输入**：完结小说全量解析为世界观知识库
2. **默认玩法**：完结后的全新故事（自由度最大）；也支持系统推荐切入点和玩家自选起点
3. **存储**：图数据库管关系和索引 + 文件系统存实际资产，节点关联文件夹路径
4. **生成策略**：流式生成，Gap 机制控制预生成间隔，边玩边生成
5. **资产管理**：分层（基础层+叠加层）避免组合爆炸，资产池按需生成+复用
6. **角色立绘**：每角色每服装 全身六视图+脸部五视图 作为一致性锚点，情绪按 角色×服装×情绪×程度(1-10) 索引
7. **场景**：基准图 + 时间段(每4h) × 世界状态 的变体
8. **世界状态**：玩家选择触发显式状态变更（入库），隐式状态由 LLM 推理
9. **角色对话**：参考酒馆AI角色卡体系
10. **配音**：默认关闭，预留接口，后续接入 CosyVoice 等
11. **引擎**：自研 Web 引擎，TypeScript + React，前后端分离
