# 小说解析方案

## 核心思路

以**完结小说**作为输入源，全量解析后作为"世界观知识库"。

## 解析目标

- 角色设定（性格、外貌、能力、关系网）
- 世界观规则（势力、地理、能力体系等）
- 剧情时间线（关键事件、转折点）
- 场景描写（地点、氛围）
- 对话风格（各角色的说话方式、语气）

## 玩家输入模式（三层优先级）

### 1. 默认：完结后的全新故事
- 世界观全量可用，自由度最大
- 不与原著剧情冲突（原著已结束）
- 本质是"官方级同人"体验

### 2. 系统推荐切入点
- 系统分析小说后，推荐若干关键转折节点
- 玩家从某个转折点开始，走出不同的 IF 线
- 本质是"如果当时走了另一条路"

### 3. 玩家自选起点
- 玩家自由指定从哪一章开始
- 输入范围越少，分支自由度越大
- 输入范围越多，世界观约束越强

## 额外模式：原著线性体验
- 完整还原原著剧情，不做分支
- 有立绘、配音、BGM 的视觉小说体验
- 可作为解析质量的验证基准
