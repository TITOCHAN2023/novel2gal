# 角色对话与配音

## 角色卡体系

参考酒馆AI（SillyTavern）的角色卡做法，为每个角色建立完整角色卡。

### 角色卡内容
- 基础信息（姓名、年龄、身份）
- 性格描述
- 说话风格（语气、口癖、用词习惯、句式特点）
- 与其他角色的关系和态度差异（对A说话温柔，对B说话冷淡）
- 背景故事摘要
- 示例对话（few-shot）

### 存储位置
角色卡作为 md 文件存放在角色节点关联的资产文件夹中：
```
character_角色名/
  character_card.md   # 角色卡
  base/
  outfit_xxx/
  voice/
```

## 配音方案

### 当前策略
- **默认关闭**，作为可选功能模块
- 预留标准接口，方便后续接入

### 后续接入计划
- CosyVoice 等 TTS 方案逐步接入
- 每个角色的 voice/ 文件夹存放音色样本（用于 voice cloning）
- 接口设计需支持多 TTS 后端切换
