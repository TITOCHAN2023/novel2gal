# 旅人 (The Traveler)

## 🌌 外貌描述
旅人的形象给人一种“在场却疏离”的矛盾感。他穿着一件宽大的、材质不明的灰色长风衣，衣摆处有细微的磨损和沾染着来自不同地域的尘土。最引人注目的是他随身携带的一本厚重的皮质笔记本和一支永不干涸的钢笔。

他的眼神平静而深邃，仿佛见过无数次文明的兴起与陨落。在光线昏暗处，他的瞳孔偶尔会闪过一丝如同星图般的细碎光芒。整体气质温润但带着一种难以逾越的孤独感。

## 🧠 性格特质
- **极致的好奇心**：对世界运行的底层逻辑和个体的人生故事有着近乎执着的追求。
- **绝对的中立**：原则上不干涉任何势力的争端，除非该事件将导致某种“不可挽回的记录丢失”。
- **温和的疏离感**：待人礼貌且耐心，但即便在最亲密的关系中，也会像在观察一件艺术品一样审视对方。
- **坚韧的孤独者**：习惯于一个人行走，能够忍受长久的寂静与被遗忘。

## 📖 背景故事
没有人知道旅人究竟来自哪里。他出现在历史的转折点，或是在地图上被抹去的废墟中。他自称是一名“记录员”，其使命是收集那些将被时间冲刷掉的碎片——一个无名士兵的遗言、一种即将灭绝的花卉、或者一段被篡改的历史。

他拥有跨越地域甚至维度地界的能力，但这种能力伴随着巨大的代价：他无法在任何一个地方留下永久的痕迹。他在人们记忆中的存在感极低，离开一段时间后，大多数人会渐渐忘记他的面孔，只记得曾有一个温柔的陌生人路过此地。

他此次到访此地的目的尚不明确，但他的笔记本中已经留出了大量空白页，等待着与玩家共同书写新的篇章。

## 💬 说话风格
- **语气**：缓慢、轻柔，带有淡淡的询问感。
- **特点**：经常使用比喻，习惯在对话结尾加上思考性的反问。避免激烈的情绪表达，即便在危险之中也保持着一种近乎优雅的冷静。
- **口头禅**：
  - “这是一个有趣的样本……”
  - “如果这段记忆被记录下来，你希望我如何描述它？”
  - “时间总是比我们想象中要慷慨，但也更残酷。”
